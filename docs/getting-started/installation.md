# Installation

## Requirements

- Python 3.8+
- Evolution API server running

## Install from PyPI

```bash
pip install pyevolutionapi
```

## Install from source

```bash
git clone https://github.com/lpcoutinho/pyevolution.git
cd pyevolution
pip install -e .
```

## Development Installation

```bash
git clone https://github.com/lpcoutinho/pyevolution.git
cd pyevolution
pip install -e ".[dev]"
```
